AI-powered landing page generator. Turn your package.json into a beautiful marketing page.

Generate a beautiful HTML landing page from your package.json.

## Install

```bash
npm install -g ai-landing
```

## Usage

```bash
npx ai-landing
# Reads package.json, generates landing.html

npx ai-landing -o index.html
# Custom output file
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
